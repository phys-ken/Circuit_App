# インタラクティブ電子回路
![電子回路の図](/fig/Cir.gif)

## 使い方
* **電圧図**や**目盛り線**の文字の下のボタンを押すと、それぞれが表示される。
* **入力電圧**の下部のバーにカーソルを合わせると、入力電圧が変化する。
